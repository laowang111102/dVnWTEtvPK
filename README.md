# 前后端分离宠物医院管理系统 java987

## 项目概述

本项目是基于前后端分离的 Spring Boot 框架开发的宠物医院管理系统。系统旨在提供用户友好的管理界面，高效地处理宠物医院的日常业务。

## 技术栈

- **开发语言**：Java
- **后端框架**：Spring Boot、MyBatis
- **前端框架**：Vue
- **编辑器**：IDEA
- **数据库**：MySQL
- **数据库管理工具**：Navicat
- **JDK版本**：JDK 8
- **构建工具**：Maven 3+

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员功能

1. 登录
2. 个人中心（个人信息、修改密码）
3. 用户管理
4. 医生管理
5. 预约
6. 病历记录
7. 医疗器械
8. 在线咨询
9. 维修上报
10. 公告栏
11. 轮播图
12. 退出

### 用户功能

1. 注册登录
2. 个人中心（个人信息、修改密码）
3. 首页（公告栏、宠物百科、医生介绍）
4. 公告信息
5. 宠物百科
6. 医生列表
7. 医生详情
8. 医生收藏
9. 医生点赞
10. 预约
11. 在线咨询
12. 退出
